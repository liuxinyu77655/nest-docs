最后我们做下预览，这个也是递归渲染 json，只不过内容不同：

创建 Preview 组件：

## 总结

这节我们实现了试卷的回显、预览、保存。

回显就是查询 id 对应的 content，parse 为 json 渲染。

保存就是把 content 保存到数据库。

预览则是递归渲染 json，但和编辑时渲染的内容不同。

至此，试卷编辑器就完成了。
